About Wordle

Wordle is a word puzzle game where players try to guess a secret five-letter word. Each guess is evaluated based on correctness and position, with feedback provided to help players deduce the correct word.

The game consists of a five-letter word chosen randomly from a predefined set of words. Players make guesses, and the game provides feedback after each attempt, indicating if letters are correct and/or in the right position.

Wordle is a fun and challenging game that tests players' vocabulary and deduction skills. It's a great way to pass the time while also exercising your brain!
